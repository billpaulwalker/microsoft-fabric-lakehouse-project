# LinkedIn Featured Item

## Title

Microsoft Fabric Lakehouse Project

## Short Description

End-to-end Microsoft Fabric lakehouse portfolio project showing Bronze/Silver/Gold architecture, Delta tables, Python/PySpark notebook patterns, validation checks, logging concepts, curated Gold tables, and Power BI-ready dimensional modeling.

## Longer Description

This project demonstrates how legacy SQL Server, SSIS, ADF, file-based, and API-style data movement can be modernized into a Microsoft Fabric lakehouse architecture. It includes medallion data layers, sample commercial real estate source data, notebook templates, logging and watermark table designs, validation checklists, deployment guidance, and a Power BI-ready Gold model.

## What It Proves

- Microsoft Fabric lakehouse architecture
- Production-style data engineering patterns
- Python/PySpark notebook design
- Delta Lake and medallion architecture understanding
- Power BI-ready data modeling
- Senior-level thinking around validation, logging, deployment, and governance
